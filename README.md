⏰ Relógio Digital em JavaFX
📖 Sobre o Projeto

Este projeto é um Relógio Digital desenvolvido em JavaFX que exibe o horário atual do sistema em tempo real. A aplicação atualiza automaticamente os segundos utilizando animações do JavaFX, oferecendo uma interface simples, limpa e funcional.

O projeto foi criado com foco educacional, visando a prática de interfaces gráficas, manipulação de tempo e animações em Java.

🎯 Objetivo

Praticar o uso do JavaFX

Trabalhar com a API java.time

Utilizar animações com Timeline e KeyFrame

Desenvolver uma interface gráfica simples e eficiente

⚙️ Funcionalidades

Exibe o horário atual do sistema

Atualização automática a cada segundo

Formato de 24 horas (HH:mm:ss)

Interface centralizada e minimalista

🧠 Conceitos Utilizados

JavaFX (Application, Stage, Scene)

Layouts (VBox)

Animações (Timeline, KeyFrame)

API de Data e Hora (LocalDateTime, DateTimeFormatter)

Programação orientada a eventos

🖥️ Interface

Texto centralizado

Fonte ampliada para melhor visualização

Fundo branco para maior legibilidade

▶️ Como Executar
Pré-requisitos

Java JDK 8 ou superior

JavaFX configurado no ambiente

IDE (IntelliJ IDEA, Eclipse ou VS Code)

Passos

Clone ou baixe o repositório

Abra o projeto na IDE

Execute a classe Main

O relógio será exibido automaticamente
